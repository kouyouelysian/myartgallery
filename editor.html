You've reached the editor page for this website's gallery. The whole system runs using a file "data.xml" - a special "eXtendable Markup Language" type file that contains information on groups, artworks, etc. It is parsed by gallery pages to display relevant artworks and info. This page provides no hidden data and you cannot directly upload or remove artworks through it, but it helps you edit the XML file and then redirects you to the Neocities file edit window to easily paste in the new XML. Please note: you have to be logged in to Neocities in order to easily overwrite your old XML. If you're not the owner of the site, please, leave this page - you cannot influence the site or gather any otherwise hidden data here.
